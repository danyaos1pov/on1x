Следуйте шагам для настройки

3. В правом верхнем углу нажмите «+»;
4. Выберите «Вставить из буфера обмена»;
5. Подключитесь к VPN.

Инструкция Android TV

Следуйте шагам для настройки

1. Установите приложение «Happ» из Google Play;
2. Откройте приложение «Happ» на телевизоре;
3. В приложении «Happ» на телефоне, справа сверху нажмите «+»;
4. Выберите «QR-код» и отсканируйте QR-код на телевизоре;
5. Подключитесь к VPN.
